# Политика конфиденциальности (Privacy Policy)

**Последнее обновление: Март 2026 г.**

Проект **Cookie Trace** (далее «Расширение») создан с заботой о вашей цифровой приватности и безопасности. Наш главный принцип: **мы не собираем, не храним и не передаем ваши персональные данные.**

## 1. Обработка данных (Локально)
Вся логика работы Расширения выполняется **исключительно на стороне клиента** (в вашем браузере).
* **Сканирование PII:** Анализ cookie-файлов на наличие чувствительных данных (номера банковских карт, email-адреса, декодирование Base64) происходит локально. Содержимое ваших cookie никогда не отправляется на внешние серверы.
* **Машинное обучение (ML):** Эвристическая классификация рекламных трекеров осуществляется с помощью встроенной модели машинного обучения (Random Forest). Никакая телеметрия для дообучения модели не собирается.
* **Журнал аудита:** История изменений cookie-файлов сохраняется только в локальном хранилище вашего браузера (`chrome.storage.local`). Вы полностью контролируете эти данные и можете удалить их в любой момент через настройки Расширения.
* **Модуль Chameleon:** Генерация цифрового шума (Data Poisoning) происходит автономно на вашем устройстве.

## 2. Использование разрешений (Permissions)
Для обеспечения заявленного функционала Расширению требуются следующие минимально необходимые права:
* `cookies` — для чтения, мониторинга, редактирования и обфускации cookie-файлов.
* `storage` — для сохранения пользовательских настроек и локального журнала аудита.
* `notifications` — для отправки системных push-уведомлений при обнаружении скрытых PII.

## 3. Передача данных третьим лицам
Поскольку Расширение не собирает и не экспортирует никакие пользовательские данные, мы физически не можем передать, продать или предоставить доступ к вашей информации третьим лицам.

## 4. Изменения в Политике конфиденциальности
Мы оставляем за собой право обновлять данную Политику конфиденциальности. Любые изменения будут отражены в этом файле с обновлением даты.
